# ESP32 on the wheels

Sorry, it is mostly a quick draft.

## Hardware

 - NodeMCU clone with ESP32-WROOM-32
 - H-Bridge MX1616
 - 5v UBEC
 - two 18650

## Software

 - ESP Async Web Server for HTTP and WebSocket

## Connection

 - Left motor: D27 - IN4, D26 - IN3
 - Right motor: D25 - IN2, D33 - IN1
